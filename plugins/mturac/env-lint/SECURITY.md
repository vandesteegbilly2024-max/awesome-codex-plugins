# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this plugin, please report it via
GitHub Issues at https://github.com/mturac/pluginpool-env-lint/issues with the
"security" label.

## Scope

This plugin executes only locally and reads from your repository. It does
not transmit data over the network. The helper script is Python standard
library only and can be audited at
https://github.com/mturac/pluginpool-env-lint.

## Supported Versions

Only the latest released version is supported.
